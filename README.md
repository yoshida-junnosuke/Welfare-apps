# 🗣️ Kotoboard（コトボード）

絵カード交換式コミュニケーションシステムのWebアプリケーションです。
言葉でのコミュニケーションが難しい方のための、視覚的で直感的なコミュニケーションツールです。

![Kotoboard Logo](Gemini_Generated_Image_544zzj544zzj544z.png)

## ✨ 主な特徴

### 🎯 コミュニケーション機能
- **カード選択で文章作成** - 絵カードを並べて文章を作成
- **音声読み上げ** - 作成した文章を自然な日本語で読み上げ
- **高速再生** - カードタップ時にエンジンをウォームアップし、即座に再生
- **視覚的ハイライト** - 現在読み上げ中のカードを強調表示

### 🎨 カスタマイズ機能
- **3種類の画像タイプ**
  - 📱 絵文字（手軽でカラフル、600種類以上）
  - 📷 写真アップロード（実物の写真が使える）
  - ✏️ 手描き（オリジナルイラスト作成）
- **柔軟なカテゴリ管理** - 自由にカテゴリを追加・編集
- **ピン留め機能** - よく使うカードを上部に固定
- **使用頻度カウント** - 自動で使用回数を記録

### 👶 2つのモード
- **子どもモード（ひらがな）** - ひらがな表示で見やすい
- **大人モード（漢字）** - 漢字表示でより詳細に

### 📊 データ管理
- **エクスポート/インポート** - JSONファイルで簡単バックアップ
- **複数のエクスポートオプション**
  - すべてのカード
  - ピン留めカードのみ
  - 選択したカテゴリのみ
- **オフライン動作** - データはブラウザに保存

### 🔧 便利な機能
- **5段階のカードサイズ** - 極小・小・中・大・特大から選択可能
- **並び替え** - デフォルト、名前順、カテゴリ順、使用頻度順
- **ピンフィルター** - ピン留めカードだけを表示
- **レスポンシブデザイン** - スマホ、タブレット、PCに対応

## 🚀 使い方

### 初めての起動

1. **ブラウザで開く**
   - iPhoneの場合：Safari推奨
   - Androidの場合：Chrome推奨
   - PCの場合：Chrome、Edge、Firefox等

2. **初期カードを確認**
   - アプリには最初からサンプルカードが用意されています
   - そのまま使うことも、編集・削除も可能です

### カードの作成

1. **「作る」ボタンをタップ**
2. **画像を選択**
   - 絵文字：プリセットから選択（600種類以上）
   - 写真：カメラで撮影 or ギャラリーから選択
   - 描く：指やペンで自由に描画
3. **文字を入力**
   - **漢字**：漢字モードで表示される文字
   - **読み上げ（ひらがな表示）**：ひらがなモード表示と音声読み上げ用のテキスト
4. **カテゴリを選択**
5. **「保存」をタップ**

### 文章の作成と読み上げ

1. **カードをタップ** - 下部のセンテンスストリップに追加
   - 最初のカードタップ時に音声エンジンがウォームアップされます
2. **順番に並べる** - 文章の語順に配置
3. **「おはなし」ボタン** - 音声で読み上げ（即座に開始）
4. **修正する場合**
   - カードをタップ：その位置から削除
   - 「1つ消す」：最後のカードを削除
   - 「全消し」：全部クリア

### カテゴリ管理

#### デフォルトカテゴリ
アプリには以下のカテゴリが最初から用意されています：
- ⭐ **必要** - トイレ、飲む、休むなど基本的な要求
- 🏠 **生活** - おはよう、ありがとうなど日常の言葉
- 🍎 **食事** - 食べ物、飲み物
- ⚽ **遊び** - 遊び、場所、活動
- 😊 **気持ち** - 感情表現
- 🎨 **色** - 色の名前
- 📦 **その他** - 分類されないもの

#### カテゴリの追加・編集
設定メニューから自由にカテゴリを管理できます。
デフォルトカテゴリは保護されており、削除できません。

## 📱 インストール方法

### GitHub Pagesで公開する場合

1. **GitHubアカウント作成**
   - [GitHub](https://github.com)で無料アカウント作成

2. **リポジトリ作成**
   - 「New repository」をクリック
   - 名前を入力（例：`kotoboard`）
   - 「Public」を選択
   - 「Create repository」

3. **ファイルアップロード**
   - 「uploading an existing file」をクリック
   - `index.html` と `Gemini_Generated_Image_544zzj544zzj544z.png` をドラッグ&ドロップ
   - 「Commit changes」

4. **GitHub Pages有効化**
   - Settings → Pages
   - Source: 「Deploy from a branch」
   - Branch: 「main」→「/ (root)」
   - 「Save」

5. **アクセス**
   - `https://ユーザー名.github.io/kotoboard/`
   - 数分後に利用可能

### iPhoneでアプリのように使う

1. **Safariで開く**
2. **共有ボタン（□↑）をタップ**
3. **「ホーム画面に追加」**
4. **名前を入力して「追加」**
5. **ホーム画面にアイコンが追加されます！**

## 💾 データのバックアップ

### エクスポート（保存）
1. 設定ボタン（⚙️）をタップ
2. 「エクスポート」を選択
3. エクスポート範囲を選択
4. ダウンロードされたJSONファイルを保管

### インポート（復元）
1. 設定ボタン（⚙️）をタップ
2. 「インポート」を選択
3. 保存したJSONファイルを選択

## 🎨 カスタマイズのヒント

### 効果的なカード作成
- **写真は実物を使う** - 認識しやすい
- **2つの文字を使い分け**
  - 漢字：大人が見やすい正式な表記
  - 読み上げ：自然に聞こえる音声用の文字（ひらがなモード表示にも使用）
- **読み上げはひらがな** - 自然な発音に
- **カテゴリで整理** - 探しやすく
- **よく使うものはピン留め** - すぐアクセス

### 文章例
- 「おかあさん」+「おやつ」+「がほしい」
- 「トイレ」+「いきたい」
- 「おんがく」+「きく」+「がほしい」

## 🔧 トラブルシューティング

### 音声が再生されない
- **デバイスの音量を確認**
- **サイレントモード解除**（iPhone）
- **ブラウザの権限確認**
- **別のブラウザで試す**
- **最初にカードをタップ** - 音声エンジンのウォームアップが必要です

### カードが表示されない
- **カテゴリフィルター確認** - 「すべて」を選択
- **ページを再読み込み**
- **ブラウザキャッシュをクリア**

### データが消えた
- **同じブラウザで開いているか確認**
- **プライベートモードではないか確認**
- **バックアップから復元**
